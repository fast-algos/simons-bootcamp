Spectral Properties

Spectral Theorem

Every \(n\times n\) symmetric matrix \(M\) has \(n\) orthonormal eigenvectors.

i.e. \({\psi}_1,\ldots,{\psi}_n\in \mathbb{R}^n\) such that they are

  • eigenvectors, \(\quad M\psi_i = \lambda_i \psi_i\) \(\qquad \lambda_1 \le \ldots \le \lambda_n\)
  • orthonormal, \(\quad \psi_i^{\top} \psi_j = 1\) if \(i=j, \quad\) 0 otherwise

Furthermore, \[{M}=\sum\limits_{i}\lambda_{i}{\psi}_{i}{\psi}_{i}^{\top} = U\Lambda U^{\top}\]

Simple Properties of Laplacians

Positive-Semidefinite (PSD)

All eigenvalues of \(L\) are \(\ge 0\)

\[\psi^{\top} L \psi = \lambda \| \psi \|_2^2 \ge 0 \]

Denoted as \(L \succeq 0\)

Positive-Semidefinite (PSD)

More generally, we denote \(A \succeq B\) iff

all eigenvalues of \(A-B\) are \(\ge 0\)

 

Equivalently, if forall \(x,\) we have \(x^{\top} Ax \ge x^{\top} Bx\)

Kernel

\(L\) always has a kernel containing the all 1’s vector

\[L{\mathbf 1} = (D-A){\mathbf 1} = {\mathbf 0}\]

Each row sums to \(wdeg\)

Thus, smallest eigenvalue of \(L\) is 0

Kernel

Is this the entire kernel? Can there be more zero eigenvalues?

\[ 0 = x^{\top}Lx = \sum_{(u,v) \in E} (x_u - x_v)^2 \]

\(x_u = x_v\) for every edge \((u,v)\)

\(x\) is constant on every connected component

Number of zero eigenvalues = Dimension of kernel = number of connected components

How large can an eigenvalue of \(L\) be?

\[\begin{align} x^{\top}Lx & = \sum_{(u,v) \in E} w_{(u,v)}(x_u - x_v)^2 \\ & \le 2 \sum_{(u,v) \in E} w_{(u,v)}(x_u^2 + x_v^2) \\ \end{align}\]
